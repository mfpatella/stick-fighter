# Asset Credits

The current art polish pass uses a small curated subset of public Kenney assets.

## Kenney UI Pack

- Source: https://kenney.nl/assets/ui-pack
- License: Creative Commons CC0
- Used for: `Kenney Future` and `Kenney Future Narrow` UI fonts.

## Kenney Background Elements

- Source: https://kenney.nl/assets/background-elements
- License: Creative Commons CC0
- Used for: sky, clouds, hills, mountains, trees, houses, fences, temple, and grass level dressing.

Only selected files are copied into `public/assets/vendor/kenney/` so the repo
stays light. The original `License.txt` files are stored beside each copied
asset subset.
